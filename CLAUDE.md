# 맹모여지도 프로젝트 가이드

## 서비스 소개
아이의 성향을 진단해서 교육 환경이 좋은 동네를 AI로 추천해주는 서비스.
대상: 자녀 교육에 관심 있는 30~40대 학부모.

## 기술 스택
- Next.js 15 (App Router)
- Tailwind CSS v4
- TypeScript

## 톤앤매너

### 레퍼런스
`references/` 폴더 참고 (YAYoung 앱)

### 컬러 팔레트
- Primary: 딥 포레스트 그린 계열 (`#1a3d2b`, `#2d6a4f`)
- Secondary: 크림/오프화이트 (`#f5f0e8`, `#faf7f2`)
- Accent: 따뜻한 골든 옐로우 또는 테라코타
- 배경: 오프화이트 또는 아이보리 계열
- 기존 파란 계열(blue-500, indigo-600)은 지양

### 타이포그래피
- 제목: 굵고 개성 있게 (font-extrabold)
- 본문: 읽기 편한 중간 굵기
- 전체적으로 여백을 충분히 확보

### 일러스트 & 아이콘
- 플랫 일러스트 스타일 선호
- 자연·동네·커뮤니티 느낌의 이미지
- 이모지는 보조 수단으로만 사용

### 전체 느낌
- 따뜻하고 신뢰감 있는 (차갑고 딱딱한 느낌 지양)
- 여백이 넉넉한 미니멀 레이아웃
- 학부모가 불안하지 않고 편안하게 사용할 수 있는 UX
- 과도한 애니메이션보다 자연스럽고 부드러운 전환

## 서비스 플로우
1. 홈 (`/`) → 진단 시작
2. 진단 (`/diagnosis`) → 시도 선택(최대 2개) → 20문항 응답
3. 결과 (`/result`) → 성향 유형 + 선택 지역 내 추천 시군구 TOP 5

## 학원 데이터 파이프라인

### 데이터 출처
- NEIS 학원교습소정보 전국 CSV (EUC-KR, 콤마 구분)
- `data/academies_raw.csv` — 원본 (git 미포함)
- `data/academy-stats.json` — 전처리 결과 (git 포함)

### 전처리
```bash
node scripts/preprocess-academies.mjs
```
- 117,000개+ 학원·교습소 → 시도/시군구/교습계열별 집계 JSON 생성
- CSV 업데이트 시 재실행 후 `academy-stats.json` 커밋

### 교습계열명 실제 값 (NEIS 기준)
`예능(대)` / `기예(대)` / `기타(대)` / `종합(대)` / `입시.검정 및 보습` / `보통교과` / `외국어` / `직업기술` / `정보` 등

### 유형별 지역 점수 산정
- `lib/academy-scoring.ts` — 유형 키 → 교습계열 가중치 매핑
- **유형 확정 후 `TYPE_REALM_WEIGHTS` 업데이트 필요**
- `app/api/region-scores/route.ts` — 집계 JSON 읽어 시군구 점수 계산 후 반환

### 현재 매핑 (미확정 — 유형 확정 시 수정)
| 유형 키 | 주요 교습계열 |
|--------|-------------|
| explorer | 입시.검정 및 보습 ×3, 보통교과 ×2, 외국어 ×1 |
| active | 기예(대) ×3, 예능(대) ×2 |
| social | 예능(대) ×3, 기예(대) ×2, 외국어 ×1 |
| safe | 입시.검정 및 보습 ×3, 보통교과 ×2, 종합(대) ×1 |
| creative | 예능(대) ×3, 기예(대) ×2, 기타(대) ×1 |
